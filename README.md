# Flask Todo App with Login

A Flask web application with authentication and todo management.

## Features
- Login system
- Session authentication
- Add tasks
- Delete tasks
- SQLite database

## Default Login
Username: admin
Password: 1234

## Technologies Used
- Python
- Flask
- SQLite3
- HTML

## Installation

pip install flask

## Run

python app.py

Open browser:
http://127.0.0.1:5000
